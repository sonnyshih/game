# 專案簡報：寶可夢大師之路 (POKEMASTER)

## 1. 專案概述
一個為寶可夢影迷與愛好者設計的高質感一頁式登陸頁面。該網站旨在讓使用者沉浸在寶可夢的世界中，並激發他們踏上旅程，成為「寶可夢大師」的熱情。

**目標受眾：** 寶可夢影迷與愛好者。
**主要語言：** 繁體中文。

## 2. 品牌與視覺識別
*   **品牌名稱：** POKEMASTER
*   **設計系統：** Cyber-Dex ({{DATA:DESIGN_SYSTEM:DESIGN_SYSTEM_1}})
*   **視覺風格：**
    *   **主題：** 具有高科技「賽博龐克 (Cyberpunk)」美學的深色模式。
    *   **配色方案：** 深色背景 (#121317) 搭配鮮豔的青色 (#00e0ff) 強調色與霓虹漸層。
    *   **字體：** Sora (無襯線體)，呈現現代且簡潔的視覺感。
    *   **UI 模式：** 採用 8px 圓角的卡片式設計，具備細膩陰影與互動懸停效果。

## 3. 網站結構與內容
*   **頂部導覽列：** 具有背景模糊效果的固定式透明導覽列，包含：冒險旅程 (Journey)、圖鑑 (Pokedex)、數據統計 (Stats) 與社群 (Community)。
*   **首頁區 (Hero)：** 強大的「開啟你的冒險」行動呼籲 (CTA) 區塊，搭配動態漸層效果。
*   **寶可夢的世界：** 介紹區塊，闡述人類與寶可夢之間的深厚羈絆。
*   **核心體驗 (特色區塊)：** 三卡片佈局，重點展示：
    1.  **捕捉與收集：** 探索稀有寶可夢。
    2.  **進化與成長：** 解鎖新的型態與技能。
    3.  **道館挑戰：** 贏得勳章並提升排名。
*   **冒險掠影 (圖片展示區)：** 網格佈局，展示充滿電影感的寶可夢戰鬥與奇遇。
*   **邁向冠軍之路 (數據區)：** 數據視覺化區塊，展示大師成就達成率 (98%)、全球活躍訓練家 (12M+) 以及 24/7 全天候開放。
*   **頁尾 (Footer)：** 包含服務條款、隱私政策、支援中心與 API 連結，並附有社群分享圖示。

## 4. 技術要求
*   **響應式網頁設計 (RWD)：** 針對桌機與手機視圖進行全面優化。
*   **技術棧：** 純 HTML5 與 CSS3 (使用 Tailwind CSS 進行佈局與樣式設定)。
*   **互動性：**
    *   所有卡片與按鈕均具備懸停效果（縮放、光影位移）。
    *   導覽列具備磨砂玻璃 (Backdrop blur) 效果。
    *   所有狀態變換均有平滑的過渡效果。
